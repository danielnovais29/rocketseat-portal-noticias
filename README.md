## 📖 Sobre

O **Portal de Notícias** é uma página desenvolvida como desafio da formação **Fullstack** da Rocketseat.

O projeto tem como objetivo praticar conceitos fundamentais de desenvolvimento Front-end, como estruturação semântica em HTML, CSS Grid, Flexbox, responsividade e organização de layouts modernos.

---

## 🚀 Tecnologias

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Git
- GitHub

Clone o repositório:

```bash
git clone https://github.com/danielnovais29/Projeto-Projeto---Portal-de-noticias---rocketseat.git
```

Acesse a pasta:

```bash
cd Projeto-Projeto---Portal-de-noticias---rocketseat
```
